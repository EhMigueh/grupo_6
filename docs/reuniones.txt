Output

- Se muestran los USUARIOS creados de manera ALEATORIA.
- Se muestran las RECOMENDACIONES por USUARIO.
- Se muestra quienes se CONECTARON en base al ÍNDICE DE JACCARD.
- Se muestra el GRAFO DE CONEXIONES.
- Se muestran las PUBLICACIONES.
- Se muestra el USUARIO con más AMIGOS.
- Se GENERA la imagen EPS y PNG.
